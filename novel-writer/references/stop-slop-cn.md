# 去AI腔 — 中文网文语调过滤指南

> 中文网文版 stop-slop。参考 GitHub hardikpandya/stop-slop 框架，针对男频热血/系统流/科幻网文适配。
> 写作完成后加载此文件，做最终自查。

## 一、禁用口癖（直接删）

### 清嗓式开场白
值得注意的是、不得不说、值得一提的是、不可否认的是、众所周知、毫无疑问、毋庸置疑、显而易见、不难看出、令人惊讶的是、事实上、实际上、老实说、说白了、简单来说、换句话说

### 二元对比套路（最大口癖）
**不是……而是……、并不……而是……、并非……而是……、不在于……而在于……、与其说……不如说……、表面上……实际上……、看起来……其实……、不只是……更是……**
→ 直接说"而是/不如说/实际上"后面的内容

### 转场/时间废话
就在此时、就在这时、正在这时、突然、忽然、瞬间、说时迟那时快、还没等他反应过来、不等他多想、来不及多想

### 心理口癖（用户钦定禁用语）
**不禁想到、不禁想道、不由想到、心中暗想、暗自想道、只见**

### 情绪套路
心中一凛、倒吸一口凉气、瞳孔猛地一缩、一股寒意从脚底直冲天灵盖、心中一暖、鼻子一酸、不知为何

### 战斗套路
身形一闪、身影一闪、身形暴退、快如闪电、势如破竹、摧枯拉朽、不费吹灰之力

### 科幻/机甲特供
冰冷的机械音响起、一道光芒闪过、白光/金光一闪、光芒大盛、流光溢彩

## 二、禁用结构

### 二元对立结构
❌ 他追求的从来不是单纯的强大，而是能够保护身边之人的力量。
✅ 他要的力量，够强就行——强到没人能碰他身边的人。

### 否定列举 → 直接说是什么
### 设问自答 → 直接说结论
### 排比堆砌 → 换一个短句
### 然后/接着/随即流水账 → 靠动作本身推进
### 段落结尾金句综合征 → 不要段段升华

## 三、正确的写法方向

| ❌ AI腔 | ✅ 更好 |
|---------|--------|
| 他心中十分惊讶 | 他嘴里的烟掉在地上 |
| 他感到一股寒意 | 他不自觉地缩了缩脖子 |
| 他意识到情况不妙 | "操。"他骂了一声 |
| 眼前是极其壮观的景象 | 眼前五米高的机甲把半边天都遮了 |

## 四、评分卡（满50分，低于38分必须重写）

| 维度 | 问题 | /10 |

| 直接性 | 有二元结构吗？ |
| 干净度 | 能找到禁用语吗？ |
