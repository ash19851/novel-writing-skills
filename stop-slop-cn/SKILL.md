# 去AI腔 — 中文网文语调过滤指南

## 核心规则

1. **禁用口癖直接删** — 清嗓式、二元对比、时间转场、心理口水话，见一次删一次
2. **动作替代心理** — "他心里很震惊" → "他嘴里的烟掉在地上"
3. **对话替代叙述** — "他意识到不妙" → "操。"他骂了一声
4. **具体替代抽象** — "极其壮观" → "五米高的机甲把半边天都遮了"
5. **不要告诉读者该有什么感受** — 写发生了什么，不要写"令人震惊的是"

## 十大禁用语清单

### 1. 清嗓式开场白
值得注意的是、不得不说、值得一提的是、不可否认的是、众所周知、毫无疑问、毋庸置疑、显而易见、不难看出、令人惊讶的是、事实上、实际上、老实说、说白了、简单来说、换句话说、也就是说、言归正传

### 2. 二元对比（最大口癖）
**不是……而是……、并不……而是……、并非……而是……、不在于……而在于……、与其说……不如说……、表面上……实际上……、看起来……其实……、不只是……更是……、不仅仅……更……**

**规则**：直接说"而是/不如说/实际上"后面的内容，扔掉前半句。

### 3. 时间转场废话
就在此时、就在这时、正在这时、突然、忽然、瞬间、刹那间、电光火石之间、说时迟那时快、千钧一发之际、异变突生、还没等他反应过来、不等他多想、来不及多想

### 4. 心理口癖（用户禁用语）
**不禁想到、不禁想道、不由想到、暗自想道、心中暗想**

### 5. 情绪套路
心中一凛、倒吸一口凉气/冷气、浑身一震、心中一惊、瞳孔猛地一缩/骤缩/微缩、一股寒意从脚底直冲天灵盖、背后冒出一层冷汗、心中一暖、一股暖流涌上心头、鼻子一酸、不知为何

### 6. 科幻/机甲特供
冰冷的机械音响起、冰冷的电子音、不带丝毫感情的声音、一道光芒闪过、白光/金光/银光一闪、光芒大盛/万丈/骤然亮起、流光溢彩

### 7. 战斗套路
身形一闪、身影一闪、身形暴退/急退、身影如鬼魅般、快如闪电/若奔雷/迅如疾风、势如破竹、摧枯拉朽、如同砍瓜切菜、如入无人之境、不费吹灰之力、轻松写意、轻描淡写地

### 8. 副词泛滥（删一半）
非常、十分、极其、无比、相当、格外、颇为、着实、简直、简直不要太

### 9. 系统流特供
一排数据流从眼前闪过、眼前出现一个半透明的面板、半透明的虚拟面板浮现在眼前、一道信息流涌入脑海、海量信息涌入大脑、庞大信息流冲击、脑中响起一声提示音、脑海中回荡着系统的声音

### 10. 废话过渡
此刻他的心中只有一个念头、此刻所有人都意识到、这个想法在他脑海中一闪而过、他的大脑飞速运转、脑海中闪过无数个念头、大脑一片空白

## 六大禁用结构

### 1. 二元对立结构
❌ "他追求的从来不是单纯的强大，而是能够保护身边之人的力量。"
✅ "他要的力量够强——强到没人能碰他身边的人。"

### 2. 否定列举
❌ "这不是测试。不是演习。不是考核。这是一场真正的战争。"
✅ "战场。真枪实弹的战场。"

### 3. 设问自答
❌ "这意味着什么？很简单——"
✅ 直接说结论。

### 4. 排比堆砌
❌ "他没有退缩，没有犹豫，没有畏惧。"
✅ "他一步没退。"

### 5. 流水账转场
❌ "他深吸一口气，然后推开了门。接着，他看到……随即，他意识到……"
✅ 去掉"然后接着随即"，靠动作推进。

### 6. 段落结尾金句综合征
❌ 每段结尾"他握紧了拳头。这一次，他不会再输。"
✅ 偶尔可以，不要段段如此。

## 改写方向

| ❌ AI腔 | ✅ 更好 |
|---------|---------|
| 他心中十分惊讶 | 他嘴里的烟掉在地上 |
| 他感到一股寒意 | 他不自觉地缩了缩脖子 |
| 他心里很愤怒 | 他拳头捏得嘎吱响 |
| 他意识到情况不妙 | "操。"他骂了一声 |
| 他感到十分震惊 | "你再说一遍？" |
| 眼前是极其壮观的景象 | 眼前五米高的机甲把半边天都遮了 |
| 他的实力有了质的飞跃 | 以前一拳碎石，现在一拳碎墙 |

## 评分卡（每章）

| 维度 | 问题 | /10 |
|------|------|-----|
| 直接性 | 有"不是…而是…"类二元结构吗？ |
| 干净度 | 能找到"值得注意的是""不得不说"吗？ |
| 真实感 | 读者会觉得这是人写的吗？ |
| 动作化 | 心理描写多还是动作/对话多？ |
| 具体性 | 抽象形容词多还是具体细节多？ |

**标准**：满50分，**低于38分必须改**。

## 快速自查（写完逐项搜）

- [ ] 搜索"不是"——是不是二元对比？
- [ ] 搜索"而是"——同上
- [ ] 搜索"值得注意的是"——删
- [ ] 搜索"不得不说"——删
- [ ] 搜索"不禁/不由/只见"——用户禁用语
- [ ] 搜索"就在此时/就在这时"——删
- [ ] 搜索"突然/忽然"——改成具体描写
- [ ] 搜索"非常/十分/极其"——删一半
- [ ] 搜索"瞳孔/倒吸"——大概率要改
- [ ] 搜索"脑海中/心中"——改成动作
- [ ] 搜索"然后/接着/随即"——去掉一半
- [ ] 每段结尾是否都在"升华"？——改动

> 核心理念：AI喜欢"解释"和"宣告"，好网文靠"展示"和"推进"。
> 读者耐心以秒计算，删一句废话就读快一个字。
