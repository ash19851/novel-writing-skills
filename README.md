# novel-writing-skills

中文网文创作 Hermes Agent 技能集 — 用于番茄小说等平台的网文创作工作流。

## 技能列表

| 技能 | 说明 |
|------|------|
| **stop-slop-cn** | 中文网文去AI腔调过滤器 — 删除AI写作中的套路化句式、禁用口癖和结构性问题 |
| **novel-evolve** | 探索式协作写作引擎 — 故事在对话中生长，在写作中自我发现 |

## 使用方法

这些文件设计为 Hermes Agent 的 skill 文件。在 Hermes 中通过 `skill_view(name)` 加载使用。

也可直接阅读 SKILL.md 文件内容作为参考。
