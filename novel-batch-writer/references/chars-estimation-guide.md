# 中文字数快速估算指南

## 核心换算关系

```
UTF-8 编码下：
  1 个汉字 ≈ 3 字节
快速估算：bytes / 3 ≈ 汉字数
```

## 常见字节数与对应字数

| 文件大小(bytes) | 大约汉字数 | 是否达标2500 |
|-----------------|-----------|-------------|
| 3000 | ~1000 | ❌ 严重不足 |
| 4500 | ~1500 | ❌ 不足 |
| 6000 | ~2000 | ❌ 偏少 |
| 7500 | ~2500 | ⚠️ 临界值 |
| 9000 | ~3000 | ✅ 达标 |

## 为什么初版就要写够

按"先写1200+再扩充到2500"的模式：
- 每章需要1-3次额外工具调用
- 60章整卷 = 60-180次额外调用

正确做法：一次写7500+ bytes，一次通过核验门。
