# 字数不足时的标准扩充流程

## 字数核验命令

```bash
grep -v '【本章完】' "8-正文-第{N}章.txt" | tr -d '[:space:]' | wc -m
```

若结果 < 2500，执行以下流程。

## 扩充流程

### Step 1：移除结束标记
```bash
sed -i 's/【本章完】//g' "8-正文-第{N}章.txt"
```

### Step 2：追加内容
使用 terminal heredoc 追加新段落，从大纲情节点中选择一个或多个展开描写。

```bash
cat >> "8-正文-第{N}章.txt" << 'EXPAND'
（新段落内容）
EXPAND
```

### Step 3：重新添加结束标记
```bash
echo '【本章完】' >> "8-正文-第{N}章.txt"
```

### Step 4：验证字数
```bash
chars=$(grep -v '【本章完】' "8-正文-第{N}章.txt" | tr -d '[:space:]' | wc -m)
```

### Step 5：如果仍不足，回到 Step 1

## 扩充内容来源优先级

1. 大纲情节点扩写
2. 人物交互细节（对话、动作、反应）
3. 环境与氛围描写
4. 技术操作描述
5. 心理活动（符合口癖规则的前提下）

## 注意事项

- 不添加大纲外的新角色、新情节、新设定
- 保持行文风格一致
- 遵守防漂移规则
