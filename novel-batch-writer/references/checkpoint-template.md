# Batch Checkpoint Template (Quick Reference)

## Summary Format (Batch Continuation Mode)

```
【核验】第{N}-{N+4}章 [剧情单元名]
覆盖率: {n}/100 ✅
基础合规: ✅/❌ | 分卷一致: ✅/❌ | 钩子链: {n}/5 ✅ | 爽点多样: ✅/❌ | 节奏: ✅/❌
综合: ✅ 通过 / ❌ 退回
```

## Decision Rules

| Condition | Action |
|-----------|--------|
| 剧情单元覆盖率 < 85 | **退回修改，不得保存** |
| 任一章出现"遗漏" | **退回修改，不得保存** |
| 分卷大纲一致性"不通过" | **退回修改，不得保存** |
| 钩子衔接链出现"断裂" | **退回修改，不得保存** |
| 全部通过 | 批准保存 |
