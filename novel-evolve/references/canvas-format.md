# Novel Canvas — 活文档系统参考

## 文件结构

项目根目录下创建 `.canvas/` 文件夹：

```
.canvas/
├── world/
│   ├── core.md
│   ├── atmosphere.md
│   └── locations/
│       ├── index.md
│       └── 海滨城市.md
├── characters/
│   ├── index.md
│   ├── 主角.md
│   └── 配角_A.md
├── plot/
│   ├── events.md
│   ├── units.md
│   └── chapters.md
├── meta/
│   ├── direction.md
│   ├── built-blocks.md
│   └── session-status.md
└── index.md
```

## 文档格式规范

### 地点文档示例
```markdown
# 海滨城市

状态：已细化（可继续进化）

## 核心特征
- 位于大陆东海岸，亚热带气候
- 支柱产业：港口贸易 + 渔业 + 旅游业

## 气候细节累积
- [首次提及] 空气中常年有海风的咸味
- [展开时] 夏季多台风，居民房屋普遍装有木板百叶窗
```

### 角色文档示例
```markdown
# 陈北 主角

状态：已进化（持续更新中）

## 初始轮廓
- 身份：28岁，港口机械维修工
- 核心驱动：想离开这座小城但不知道自己能去哪

## 进化记录
- [第1章] 回避眼神接触 → 追加社交特征
- [第3章] 应激冷冻 → 追加压力反应
```

### 事件链文档示例
```markdown
# 事件链

### 事件1：意外的信（已完成）
- 起因：主角在维修室发现一封没有署名的信
- 落点：主角决定去码头找人问清楚
- 遗留问题：谁写的信？
```

### session-status 文档（精简版）
```markdown
# 会话状态

当前阶段：Phase 4（写作循环）
已完成积木：全部6块 ✅
当前事件：码头夜谈（第3-5章）
最新写作：第3章 ✅ (2830字，含回填)
更新时间：[自动]
```
